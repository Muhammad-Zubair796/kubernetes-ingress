DevOps Project by M Zubair

Project Agenda: Kubernetes Deployment

This project demonstrates a fully containerized Python application running on a production-ready environment.

Technical Implementation:

  • Docker: Used for containerizing the Python web application into a portable image.
  • Minikube: Orchestrating the local Kubernetes cluster on an AWS EC2 t3.micro instance.
  • Deployments: Managing 2 replicas of the application to ensure high availability and self-healing.
  • Services: Using a NodePort Service to load balance traffic across pods and expose the app to the internet.

This setup showcases the complete CI/CD flow from code to a running Kubernetes cluster.
